# HerpicusJS
Herpicus Javascript Library

### Type Functions
**Herpicus.TypeOf**   
Return a *string* type for any operand
```
Herpicus.TypeOf({}) //object
Herpicus.TypeOf(document.createElement('div')) //htmldivelement
```
